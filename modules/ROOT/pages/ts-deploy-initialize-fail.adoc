= Deployment Initialization failed

ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

== Cause

Temporary System or connectivity issues between the Partner Manager control plane and the deployment server or (in case of CloudHub deployment) Anypoint MQ. 

== Solution

* Verify that your deployment server group *B2BServerGroup* is in the `Running` state.

For On-Premises, try redeploying the message flow.

* Try redeploying the message flow after 10 minutes.

If the error persists, create a support ticket with MuleSoft along with the Reference number in the deployment error.
