= Deployment Initialization failed

ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

Causes:
Temporary System / connectivity issues between the Partner Manager control plane and the deployment server, or Anypoint MQ (in case of CloudHub deployment)

Solution
Verify the health of your deployment server group (B2BServerGroup) to make sure it is in Running state try redeploying the message flow ( On Premise Only)
Try redeploying the message flow after 10 minutes.
If the error persists, please create a support ticket with MuleSoft along with the Reference number in the deployment error.
