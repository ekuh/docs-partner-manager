= Control Numbers Settings
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

Any of these options with a checked, 'yes', or 'on' setting means that certain control number settings can be blocked. For any of the interchange, group, and transaction set numbers that are set to 'yes' or 'on', a duplicate transmission check can be set to a minimum of 1 day to a maximum of 60 days. The default is 30 days.

[%header,cols="3s,7a"]
|===
|Setting Name and Description |Default

|Requires Unique Interchange Control Number (ISA13)
|Checks that the received interchange does not match the Unique Interchange Control Number (ISA13). If a match is detected, the duplicate interchange is blocked.

Default::

* `On`

|Setting Name and Description |Default

|Requires Unique Group Control Number (GS06)
|Checks that the received interchange does not match the Unique Group Control Number (GS06). If a match is detected, the duplicate group control number is blocked.

Default::

* `Off`

|Setting Name and Description |Default

|Requires Unique Transaction Set Control Number (ST02)
|Checks that the received interchange does not match the Transaction Set Control Number (ST02). If a match is detected, the duplicate transaction set control number is blocked.

Default::

* `Off`

|===
